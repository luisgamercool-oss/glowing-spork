# glowing-spork
The glowing-spork we all know and love with the quality-of-life features of the new launcher. This aims to support java 8 through modern java.

## Features
- Microsoft authentication
- Bundled JVMs
  - Automatically downloads the appropriate JVM for all minecraft versions
  - You just need a runtime to open the actual launcher
  - You can still provide your own JVMs
- Update checking
